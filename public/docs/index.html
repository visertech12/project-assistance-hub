Installation Guide


# Clone the repository
git clone https://github.com/your-username/codemarket.git

# Navigate to project directory
cd codemarket

# Install all dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
